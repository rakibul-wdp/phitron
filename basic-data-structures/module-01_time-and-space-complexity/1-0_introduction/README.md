### Time and Space Complexity

- Time and space complexity
- O(1), O(logN), O(sqrt(N))
- O(N), O(NlogN), O(N\*N)

#### Time Complexity Limit

- O(1) ->
- O(logN) -> 10^18
- O(sqrt(N)) -> 10^14
- O(N) -> 10^7
- O(NlogN) -> 10^5
- O(N^2) -> 10^3

#### Space Complexity Limit

- O(N) -> 10^6
- O(N^2) -> 10^3
- O(NlogN) -> 10^5 -> 1.6 \* 10^6
- O(logN) -> 10^18
- O(sqrt(N)) -> 10^12

#### Ascending order

- O(1) -> O(log N) -> O(sqrt(N)) -> O(N) -> O(N log N) -> O(N^2) -> O(2^N)

## Rules for naming variables

- Variable name must start with a letter or underscore
  - abul (right)
  - \_abul (right)
  - @abul (wrong)
  - 123abul(wrong)
- Variable name must contain letter, digits or underscore
  - abul123
  - abul_123
  - abul 123 (wrong)
- Keywords can't be used as variables
  - int (wrong)
  - for (wrong)
  - float (wrong)
